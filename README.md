# Липин Никита БАС1 Лабораторная No9

# Задание 1 

## Задача 1 

### Текст задачи

Реализовать приложение из ЛАБОРАТОРНОЙ РАБОТЫ №6 ЗАДАНИЯ 2 и 3 (Перегрузка операций) в
виде WPF-приложения с графическим интерфейсом. Все требования к функционалу сохраняются.

### Алгоритм решения задачи

Класс MainWindow
 - конструктор с инициализацией
 - метод для взятия данных из полей
 - метод для отправки сообщения в приложение
 - метод обработки запроса площади 1 треугольника
 - метод обработки запроса ToString для треугольников
 - метод обработки запроса инкримента для 1 треугольника
 - метод обработки запроса декримента для 1 треугольника
 - метод обработки запроса на сравнение треугольников
 - метод обработки запроса на преобразование треугольника
Класс Checker
 - Метод проверки double
XAML MainWindow
 - поля катетов треугольников
 - кнопки для операций
 - консоль для вывода действий
